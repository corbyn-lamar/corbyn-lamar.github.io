Tool Purpose

This tool is designed to streamline the creation of procedurally generated portions of levels for 3D games. It is focused on enhancing the workflow for generating connected room systems, such as dungeons. It will feature options for generating in an area, room placement, room adjustment, hall placement, staircase placement, and other features for how it generates. This tool is useful for placing procedural dungeons in between some of the more handcrafted set pieces.

Target Audience

The tool is intended for level designers and game developers working on procedurally generated environments in 3D games, particularly those in roguelike, dungeon crawler, or adventure genres. It is also may be useful for technical artists and designers who want more control over procedural level generation in their levels. It’s mainly a tool to connect pieces of level together through procedural dungeons rather than being a standalone dungeon.

Problem the Tool Solves

Making parts of a large and re-playable level can be difficult. This tool allows developers to generate connected, cohesive, and traversable dungeons procedurally without super extensive manual placement. This tool should simplify the process by providing intuitive features for dungeon generation, improving the speed and efficiency of level creation. Developers can use this tool to generate diverse dungeon layouts in a streamlined and efficient manner. The tool will be focused mainly on generating a procedural portion of a level between set pieces that the level designers could have.

Vision Statement

The tool's vision is to create a modular, flexible, and intelligent way of procedurally generating a portion of a level. The design principles revolve around accessibility for non-programmers, ease of integration into existing projects, and adaptability/modularity for various game mechanics. The tool should allow for iterative adjustments and scalability for various game sizes and complexities.
